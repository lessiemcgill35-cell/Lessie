# How to Download and Play Pepper Dash

## 1) Download the game

### Option A: Clone with Git

```bash
git clone <REPOSITORY_URL>
cd Lessie
```

### Option B: Download ZIP

1. Open the repository page in your browser.
2. Click **Code**.
3. Click **Download ZIP**.
4. Extract the ZIP file.
5. Open a terminal in the extracted `Lessie` folder.

---

## 2) Start the game locally

Run a local web server from the project folder:

```bash
python3 -m http.server 8000
```

Then open one of these URLs in your browser:

- `http://localhost:8000/app.html` (full-window app launcher)
- `http://localhost:8000/index.html` (direct game page)

---

## 3) Controls

- **A / D**: Move
- **Space**: Jump
- **Shift**: Dash
- **E**: Taunt

---

## 4) Goal

Collect pizza slices, avoid or dash through enemies, and reach the golden finish door.
